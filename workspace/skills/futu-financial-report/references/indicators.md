# 财务指标说明

## 利润表指标

### 收入指标

| field_id | 指标名称 | 说明 |
|----------|----------|------|
| 8001 | 总收入 | 营业收入总额 |
| 8002 | 总收入 YoY | 同比增长率 |

### 成本指标

| field_id | 指标名称 | 说明 |
|----------|----------|------|
| 8003 | 营业总成本 | 营业成本总额 |
| 8004 | 毛利 | 收入减去营业成本 |
| 8010 | 营业总成本 YoY | 同比增长率 |

### 费用指标

| 8018 | 营业利润 YoY | 同比增长率 |

| 8035 | 所得税 | 所得税费用 |

| 8044 | 净利润 YoY | 净利润同比增长率 |

## 财务比率

| 比率名称 | 计算公式 | 说明 |
|----------|----------|------|
| 毛利率 | 毛利 / 总收入 × 100% | 衡量核心盈利能力 |
| 费用占比 | 营业费用 / 总收入 × 100% | 衡量费用控制能力 |
| 营业利润率 | 营业利润 / 总收入 × 100% | 衡量营业效率 |
| 净利润率 | 净利润 / 总收入 × 100% | 衡量整体盈利能力 |

## YoY数据处理规则

1. 如果API返回的YoY值 < 1：视为小数形式，乘以100转为百分比
2. 如果API返回的YoY值 >= 1：直接保留（已经是百分比形式）
3. 如果YoY为负数：保持负数

## 金额单位转换

- 原始数据单位：千元
- 目标单位：百万元
- 转换公式：百万元 = 千元 / 1000
