## 加工指标

### 自由现金流 FCF (Free Cash Flow)

**公式：**

```
FCF = (调整后) 经营活动现金净流入 − 经营性资本开支
    = 经营活动现金流量净额 − 资本支出
```

两项均来自现金流量表，是各市场必有字段，可跨 A 股、港股、美股稳定计算。

**期间对齐（v0.7.8+）**：现金流量表默认按季度维度拉取（HK/A股 `financial_type=11`
累计季报，美股 `financial_type=10` 单季+年报），与利润表期次严格一致。因此 FCF 的
每一期分子（OCF）与减项（CapEx）都来自同期次现金流量表。

例如：2026Q1 利润表（1-3 月累计）↔ 2026Q1 现金流量表（1-3 月累计），FCF 直接反映
该季度累计的自由现金流；2026FY 利润表 ↔ 2026FY 现金流量表，反映全年 FCF。

**各市场字段映射：**

| 市场 | OCF (现金流表) | 资本支出 (现金流表) |
|------|---------------|-------------------|
| A股  | 3001 经营活动现金流量净额 | 3043 购建固定资产、无形资产和其他长期资产支付的现金 |
| 港股 | 5001 经营活动现金流量净额 | 5071 购买固定资产 + 5073 购买无形资产（累计季报缺失时回退 5069 投资活动净额） |
| 美股 | 8015 经营活动现金流量净额 | 8046 固定资产交易净额（缺失时回退 8042 投资活动净额） |

任意期次 OCF 或 CapEx 缺失时该期 FCF 显示为空。

**衍生展示指标 - 经营活动现金流量净额 OCF**

利润表 Excel 会在 FCF 行之前展示一行 `经营活动现金流量净额`，展示公式分子 OCF 的原始值
（单位与 FCF 一致），便于对照 `FCF = OCF - CapEx` 的推导过程。数据来源与 FCF 一致，
均取自同期次现金流量表；若该期现金流量表无对应数据则显示为空。

**衍生展示指标 - 资本开支 CapEx**

利润表 Excel 中在 `经营活动现金流量净额` 与 `自由现金流(FCF)` 两行之间展示一行
`资本开支(CapEx)`，直接展示上述公式中 CapEx 分项的值（已取绝对值，单位与 FCF 相同），
方便对照 `FCF = OCF - CapEx` 的计算过程。若某期只有 CapEx 单独可用（如 OCF 缺失导致
FCF 为空），CapEx 仍会独立展示。

**衍生比率 - 自由现金流净利润比 FCF/NI**

紧跟 FCF 行之后，公式：

```
FCF/NI = FCF / 净利润 × 100%
```

分母口径与 FCF 的"公司整体"口径对齐：优先使用**总净利润**（US `8037` / HK `5045` /
A股 `3043`），缺失时回退到**归母净利润**（US `8043` / HK `5051` / A股 `3047`）。

**语义**：
- `>100%`：现金创造能力强于账面利润，通常见于高折旧/客户预付款/递延收入业务
- `~100%`：账面利润基本能转化为自由现金流，盈利质量健康
- `<100% 或负值`：多因大额 CapEx、应收账款拉长、存货堆积等现金流质量问题，需要结合
  CapEx / 营运资本变动进一步分析

### 总资产收益率 ROA / 净资产收益率 ROE

**公式：**

```
ROA = 年化净利润 / 资产合计 × 100%
ROE = 年化归属母公司净利润 / 归属母公司股东权益合计 × 100%
```

**期间对齐与年化：**

- 分子来自利润表（interim 期次需年化）：FY×1、H1/Q6×2、Q9×4/3、Q1/Q2/Q3/Q4×4
- 分母来自资产负债表（季度末时点快照）
- **周期一致性（v0.7.8+）**：资产负债表默认与利润表同频拉取（HK/A股 `financial_type=11`
  累计季报，美股 `financial_type=10` 单季+年报），并按 `period_text` **精确匹配**每一期。
  例如：2026Q1 利润表 → 2026Q1 资产负债表（2026-03-31 快照），周期严格一致
- 若某期利润表在资产负债表中找不到同期次（如富途未披露该季 BS），依次回退：
  ① 同 fiscal_year 的年报余额 → ② 邻近历史 BS → ③ 首条 BS 兜底
- 分母字段按候选列表顺序回退，以适应不同公司字段命名差异（如 MSFT 使用 8081 而非 8100）

**各市场字段映射：**

| 市场 | 资产合计 | 归属母公司股东权益合计（候选顺序） | 归母净利润 | 净利润 |
|------|--------|----------------------------------|----------|-------|
| A股  | 3001   | 3098 → 3097                       | 3047     | 3043  |
| 港股 | 5001   | 5110 → 5109                       | 5051     | 5045  |
| 美股 | 8001   | 8085 → 8081 → 8100 → 8101         | 8043     | 8037  |

**副产物：** 生成利润表时若 `workspace/excels/` 下无 7 天内的同代码资产负债表 Excel，
则自动生成一份季度维度的资产负债表（复用刚拉取的 BS 数据，不重复调用 API）。

## YoY数据处理规则

1. 如果API返回的YoY值 < 1：视为小数形式，乘以100转为百分比
2. 如果API返回的YoY值 >= 1：直接保留（已经是百分比形式）
3. 如果YoY为负数：保持负数

## 金额单位转换

- 原始数据单位：千元
- 目标单位：百万元
- 转换公式：百万元 = 千元 / 1000
