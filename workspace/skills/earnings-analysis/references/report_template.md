### 格式要求

#### 字体和排版
- 标题：Arial 14pt 加粗
- 副标题：Arial 12pt 加粗
- 正文：Times New Roman 11pt
- 表格：Arial 9pt，表头加粗、灰色背景
- 图表标题：Arial 10pt 居中
- 图表来源：Arial 8pt 居中

#### 颜色使用
- 正向数据：绿色（RGB 0, 128, 0）
- 负向数据：红色（RGB 192, 0, 0）
- 中性数据：黑色
- 强调：蓝色（RGB 0, 0, 255）

#### 超链接格式
- 所有URL必须格式为可点击超链接
- 显示文本有意义（如"Q3 2024 Earnings Release"）
- 蓝色下划线文本，Ctrl+Click打开
- 示例：`[Q3 2024 Earnings Release](https://investor.company.com/news/q3-2024)`

#### 图表规范
1. 所有图表必须有编号和标题
    - 格式：`Figure X - [图表标题]`
2. 所有图表必须有来源
    - 格式：`Source: [来源描述]`
3. 图表风格统一
    - 相同指标使用相同颜色
    - 包含图例（如有多条线）
    - 坐标轴标签清晰
4. 图表尺寸适当
    - 宽度：4-6英寸
    - 高度：3-4英寸

#### 表格规范
1. 所有表格必须有来源行
    - 表格底部灰色小字
2. 列标题加粗并灰色背景
3. 数字格式统一
    - 货币：$XX,XXX 或 $XX.XX
    - 百分比：XX.X% 或 +XXbps
    - 增长率：+X.X% 或 -X.X%
4. 对齐方式
    - 文本左对齐
    - 数字右对齐
    - 百分比小数点对齐
