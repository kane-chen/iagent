## 数据字典

### Excel文件结构说明

#### 资产负债表

| 字段名 | 描述 | 单位 | 计算逻辑/备注 |
|--------|------|------|---------------|
| 资产合计 | 公司总资产 | 百万美元 | 流动资产合计 + 非流动资产合计 |
| 流动资产合计 | 流动资产总额 | 百万美元 | 现金及等价物 + 应收款项 + 其他流动资产 |
| 现金及现金等价物和短期投资 | 流动性最强的资产 | 百万美元 | 现金和现金等价物 + 短期投资 |
| -现金和现金等价物 | 库存现金和银行存款 | 百万美元 | 直接读取 |
| -短期投资 | 购买日期少于1年的投资 | 百万美元 | 直接读取 |
| 应收款项 | 客户欠款 | 百万美元 | 直接读取 |
| -应收账款净额 | 扣除坏账准备后的应收账款 | 百万美元 | 直接读取 |
| 固定资产净额 | 扣除折旧后的固定资产 | 百万美元 | 固定资产 - 累计折旧 |
| 总投资 | 投资性资产总额 | 百万美元 | 可供出售证券 + 其他投资 |
| 归属于母公司股东权益合计 | 母公司所有者权益 | 百万美元 | 资产合计 - 负债合计 |
| 股本 | 发行在外的股份价值 | 百万美元 | 直接读取 |

#### 现金流量表

| 字段名 | 描述 | 单位 | 计算逻辑/备注 |
|--------|------|------|---------------|
| 经营活动现金流量净额 | 主要经营产生的现金流 | 百万美元 | 直接读取 |
| 折旧摊销及损耗 | 非现金支出调整 | 百万美元 | 直接读取 |
| 投资活动现金流量净额 | 投资、资本开支等现金流 | 百万美元 | 直接读取 |
| 融资活动现金流量净额 | 借款、分红、回购等现金流 | 百万美元 | 直接读取 |
| 自由现金流 | 可自由分配给债权人和股东的现金流 | 百万美元 | 经营活动现金流量净额 - 资本开支 |

#### 利润表

| 字段名 | 描述 | 单位 | 计算逻辑/备注 |
|--------|------|------|---------------|
| 总收入 | 公司总收入 | 百万美元 | 直接读取 |
| 毛利 | 收入减去销售成本 | 百万美元 | 总收入 - 营业总成本 |
| 毛利率 | 毛利占收入的百分比 | % | (毛利 / 总收入) * 100 |
| 营业利润 | 核心经营业务利润 | 百万美元 | 毛利 - 营业费用 |
| 营业利润率 | 营业利润占收入的百分比 | % | (营业利润 / 总收入) * 100 |
| 净利润 | 最终利润 | 百万美元 | 直接读取 |
| 净利润率 | 净利润占收入的百分比 | % | (净利润 / 总收入) * 100 |
| ROA(总资产收益率, 年化) | 资产盈利能力指标 | % | (净利润 / 资产合计) * 100 |
| ROE(净资产收益率, 年化) | 权益盈利能力指标 | % | (归属于母公司股东净利润 / 归属于母公司股东权益合计) * 100 |

### 关键计算逻辑

1. **增长率计算**
    - 环比增长率 = (本期值 - 上期值) / 上期值 × 100%
    - 同比增长率 = (本期值 - 去年同期值) / 去年同期值 × 100%

2. **利润率计算**
    - 毛利率 = 毛利 / 总收入 × 100%
    - 营业利润率 = 营业利润 / 总收入 × 100%
    - 净利润率 = 净利润 / 总收入 × 100%

3. **自由现金流计算**
    - 自由现金流 = 经营活动现金流量净额 - 资本开支

4. **盈利能力比率**
    - ROA = 净利润 / 资产合计 × 100%
    - ROE = 归属于母公司股东净利润 / 归属于母公司股东权益合计 × 100%

### 数据验证规则

1. **时效性验证**
    - 最新财报日期必须在当前日期3个月内
    - 如果超过3个月，发出警告

2. **完整性验证**
    - 关键指标字段不能为空或无效
    - 计算字段的结果必须合理（如利润率在0-100%之间）

3. **一致性验证**
    - 跨表格的关联数据必须一致（如净利润在利润表和现金流量表中应匹配）
    - 历史数据的连续性检查

### 数据来源与更新频率

- **数据来源**: 公司10-Q/10-K filings，投资者关系网站
- **更新频率**: 每季度财报发布后更新
- **数据覆盖**: 2020Q1至最新季度
