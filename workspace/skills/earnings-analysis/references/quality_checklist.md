
## 质量检查清单

### 数据质量检查

#### 数据时效性
- [ ] 最新财报日期已确认并记录
- [ ] 财报日期在当前日期3个月内
- [ ] 如超过3个月，已说明原因并发出警告
- [ ] 电话会议记录日期与财报发布日期一致

#### 数据完整性
- [ ] 所有关键财务指标字段存在
- [ ] 无空值或无效值（如负数在应为正数的字段）
- [ ] 历史数据连续性检查通过
- [ ] 跨表格关联数据一致（如净利润）

#### 数据准确性
- [ ] 计算字段验证（毛利率=毛利/收入）
- [ ] 增长率计算验证（环比、同比）
- [ ] 比率计算验证（ROE、ROA）
- [ ] 奇异值识别和处理

### 分析质量检查

#### 分析深度
- [ ] 超越表面数据，解释"为什么"
- [ ] 多个季度趋势分析（至少8个季度）
- [ ] 驱动因素识别（成本、价格、销量等）
- [ ] 同行业比较（如数据可用）

#### 分析逻辑
- [ ] 因果关系清晰（A导致B）
- [ ] 证据充分（数据支持结论）
- [ ] 排除替代解释
- [ ] 一致性检查（结论间不矛盾）

#### 预估合理性
- [ ] 历史趋势延伸合理
- [ ] 行业趋势考虑
- [ ] 管理层指引纳入
- [ ] 情景分析（如适用）

### 报告质量检查

#### 内容完整性
- [ ] 标题页包含所有必要信息
- [ ] 执行摘要涵盖关键点
- [ ] 详细分析覆盖所有维度
- [ ] 投资论点更新完整
- [ ] 估值和预估清晰呈现
- [ ] 来源部分完整

#### 格式规范性
- [ ] 字体、大小、颜色符合模板
- [ ] 表格格式统一（来源行、表头样式）
- [ ] 图表格式统一（编号、标题、来源）
- [ ] 超链接格式正确且可点击
- [ ] 页面布局整洁专业

#### 写作质量
- [ ] 语言简洁直接
- [ ] 避免冗余和重复
- [ ] 术语使用准确一致
- [ ] 拼写和语法无误
- [ ] 专业语调维持

### 可靠性检查

#### 来源可信度
- [ ] 主要数据来自10-Q/10-K文件
- [ ] 电话会议记录来自可靠来源
- [ ] 网络搜索信息有来源标注
- [ ] 一致预期数据注明来源和日期

#### 假设透明性
- [ ] 所有关键假设明确说明
- [ ] 预估基础和方法解释
- [ ] 敏感性分析（如适用）
- [ ] 限制条件说明

#### 偏见控制
- [ ] 承认局限性
- [ ] 平衡正反面信息
- [ ] 避免过度解读
- [ ] 区分事实和观点

### 时效性检查

#### 发布时机
- [ ] 在财报发布后24-48小时内完成
- [ ] 如延迟，说明原因
- [ ] 数据确认为最新
- [ ] 预期数据为预财报期数据

#### 信息更新
- [ ] 最新市场信息纳入
- [ ] 最新分析师观点考虑
- [ ] 同行业最新数据比较
- [ ] 最新监管变化考虑

### 错误分类和严重程度

#### 严重错误（必须修复）
1. 数据来源错误
2. 计算错误
3. 逻辑矛盾
4. 事实错误

#### 重要错误（应修复）
1. 缺少关键分析
2. 格式严重不符
3. 超链接无效
4. 拼写错误

#### 次要错误（可选修复）
1. 标点符号
2. 措辞优化
3. 布局微调
4. 样式一致性
