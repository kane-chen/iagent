# industry-stat

请创建一个agent，其可以实现行业分析，并给出投资建议：采用计划模式，先输出整体计划；然后就没有澄清的问题与用户进行问答交互；最后按照计划逐步执行，输出行业分析报告。
注意：有些skill还没有，预留placeholder。
执行步骤如下：
1、行业体量分析：调用‘行业数据skill’获取行业最近N个季度的数据，比如行业总量、渗透率等，比如电商行业相关的社零数据、网络社零数据、电商渗透率。
2、行业主要参与者：该行业主要的参与者，通过‘搜索skill’查询该行业头部公司，并预留一个用户交互问题，预留让用户补充公司。比如电商行业阿里巴巴、拼多多、京东
3、行业盈利能力分析：调用‘公司财务数据skill’获取行业主要参与者的最近N个季度的GTV/GMV、收入、毛利、运营利润率数据，以及行业对应的业务分部的GTV/GMV、收入、毛利、运营利润率数据。
4、行业竞争格局分析：基于4的数据，分析各参与者的行业占比，以及变化趋势，分析各参与者的竞争能力和盈利能力。
5、国际同行业分析：按照上述行业体量、主要参与者、盈利能力和竞争格局分析思路，对美国同行业进行分析。比如美国电商行业。
6、基于上述分析，给出投资建议：该行业后续增长趋势、各主要参与者的行业份额趋势、盈利趋势。
